Dispersive Art Gallery Problem

Introduction

The art gallery problem is a well known problem in computational geometry. The problem is described as follows : An art gallery wants to hire guardians to keep the gallery safe. They want to hire the smallest number of guards possible but be sure that together they can observe the whole gallery. There must be no blind spot. In a more formal way we can express it in this way : given a polygon P and an integer n, give a set of k guards that sees every point of the polygon if it exists.

In a more formal way, we obtain : given a polygon P and a distance l, give a set of any numbers of guards such that the distance between each guard is at least l and they see every point of the polygon if it exists. We will also consider that guards are restricted to being placed only at the vertices of the polygon. This means that while the entire interior must be visible, guards cannot be positioned arbitrarily within the polygon or along its edges. This limitation focuses the problem on vertex-based visibility.
