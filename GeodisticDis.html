Geodistic Distance

Definition:

Given two points p,q in a polygon P of n vertices, the geodesic distance is the length of the shortest path between p and q within the polygon. Unlike the Euclidean distance, the geodesic distance doesn't have to be straight. It can be broken lines.

If a polygon is convex, then the geodesic distance is equal to the euclidean distance because there are no obstacle to avoid.

There are two important concepts in geodesic distances:

  1. Geodesic Center : the Geodesic Center is a point in the polygon that minimizes the maximum geodesic distance to any other point.
  2. Geodesic Diameter: It is the maximum geodesic distance between any two points in the polygon.
Properties:
  1. Given two points p,q in a polygon P, if P is a simple polygon, then it exists a geodesic path between them and it is unique.
  2. Given two points p,q in a polygon P, the geodesic path between p and q is always a polygonal chain whose vertices (other than the endpoints) are reflex vertices of P.
  3. Given two points p,q in a polygon P, if the geodesic path between p and q is a straight line segment, we say that the points can "see" each other.
The funnel algorithm:

The Funnel Algorithm is an linear algorithm that finds the shortest path between p and q in a polygon by computing the funnel between them.

Here are the step of the algorithm :

  1. Triangulate the polygon. It can be done in linear tine.
  2. Find the τ(p) et τ(q)(the triangle in which p and q are contained).
  3. Find the τ(π(p,q)) (the set of diagonals that is crossed by the path between p and q) and sort them. (the i-th diagonal is the i-th diagonal we encounter on the path.)
  4. Initialize the funnel the first F_1 = \(α_1, p, β_1\) where α_1 and β_1 are the vertices of the first diagonal intersected by the path. This funnel represents the shortest path from p to the first diagonal.
  5. For each subsequent triangle i along the path, form F_i by adjusting the vertices to keep the shortest path from p to the current diagonal.
  6. Iterate until we reach q.

It also exists approximation algorithms but we won't discuss them.
