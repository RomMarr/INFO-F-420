Introduction

The art gallery problem is a well known problem in computational geometry. The problem is described as follows : An art gallery wants to hire guardians to keep the gallery safe. They want to hire the smallest number of guards possible but be sure that together they can observe the whole gallery. There must be no blind spot. In a more formal way we can express it in this way : given a polygon \textbf{P} and an integer \textit{n}, give a set of k guards that sees every point of the polygon if it exists. We will address the original problem but our main focus will be on a modified version of it. In this problem we don’t try to minimize the amount of guards but we want to maximize the distance between each guard. Maximizing the distance can be more efficient for the security of the gallery. In a more formal way, we obtain : given a polygon P and a distance l, give a set of any numbers of guards such that the distance between each guard is at least l and they see every point of the polygon if it exists. We will also consider that guards are restricted to being placed only at the vertices of the polygon. This means that while the entire interior must be visible, guards cannot be positioned arbitrarily within the polygon or along its edges. This limitation focuses the problem on vertex-based visibility.

Art Gallery Problem

As mentioned previously, the Art Gallery Problem is a problem where we must find the minimum number of guards that need to be placed in an n-vertex simple polygon such that all points of the interior are visible. Two points are visible to each other if the line between them is always inside the polygon. This problem is NP-Hard, meaning that it is difficult to find a solution but easy to verify one. It is challenging to find a solution because we need to consider all possible combinations of guard placements to ensure that every point in the polygon is visible. As the number of vertices increases, the number of possible configurations grows exponentially. However, verifying a solution is relatively easy, as it only involves checking that every point is visible to at least one guard. In 1973, Václav Chvátal demonstrated in a complicated proof that to guard a simple polygon with n vertices, $\lfloor\frac{n}{3}\rfloor$ guards are always sufficient and sometimes necessary. Later, Steve Fisk offered a simpler proof by using the triangulation of a polygon. Every simple polygon can be triangulated. Then, we can use the vertices from the triangulation to 3-color the graph. Every triangle must have all three colors. The color used the fewest vertices defines a valid set of guards with at most $\lfloor\frac{n}{3}\rfloor$ because the n vertices of the polygon are partitioned with 3 colors. Bellow, we have implemented the algorithm that find a set of guards such that all the points are visible using the proof above. Definition
Given two points p,q in a polygon P of n vertices, the geodesic distance is the length of the shortest path between p and q within the polygon. Unlike the Euclidean distance, the geodesic distance doesn't have to be straight. It can be broken lines.\cite{bose_funnel_algorithm_2015} If a polygon is convex, then the geodesic distance is equal to the euclidean distance because there are no obstacle to avoid. \cite{cortesi_geodesic_distances_2020} There are two important concepts in geodesic distances. \begin{itemize}[label=\textbullet] \item The Geodesic Center : the Geodesic Center is a point in the polygon that minimizes the maximum geodesic distance to any other point. \item The Geodesic Diameter: It is the maximum geodesic distance between any two points in the polygon.\cite{Abrahamsen2016} \end{itemize}
Properties
\begin{itemize}[label=\textbullet] \item Given two points p,q in a polygon P, if P is a simple polygon, then it exists a geodesic path between them and it is unique. \item Given two points p,q in a polygon P, the geodesic path between p and q is always a polygonal chain whose vertices (other than the endpoints) are reflex vertices of P \item Given two points p,q in a polygon P, if the geodesic path between p and q is a straight line segment, we say that the points can "see" each other.\cite{aichholzer_geodesic_polygon_2013} \end{itemize}
